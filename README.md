# 📌 Portfolio Website

Selamat datang di repository **Portfolio Website** saya.  
Project ini merupakan website portofolio pribadi yang dibuat untuk menampilkan profil, skill, pengalaman, project, serta kontak saya secara profesional.

## 🚀 Tentang Project

Website portofolio ini dibuat sebagai media personal branding dan dokumentasi project yang pernah saya kerjakan.  
Dengan desain yang responsif dan modern, website ini dapat diakses dengan baik di berbagai perangkat.

## ✨ Fitur

- Halaman profil pribadi
- Informasi skill dan teknologi yang dikuasai
- Showcase project
- Pengalaman kerja / organisasi
- Contact section


## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS
- JavaScript


## 📂 Struktur Project

```bash
portfolio/
│── assets/
│   ├── css/
│   ├── js/
│   ├── images/
│── index.html
│── README.md
```

## ⚙️ Instalasi & Menjalankan Project

1. Clone repository ini

```bash
git clone https://github.com/username/portfolio.git
```

2. Masuk ke folder project

```bash
cd portfolio
```

3. Jalankan project di browser

```bash
open index.html
```

Atau gunakan Live Server di VS Code.
